# People Analytics – Turnover em Instituição Financeira

## 🏦 Contexto do Projeto

Este projeto simula a atuação de um Analista de Dados em transição de Júnior para Pleno dentro de um banco de médio porte (aprox. 2.500 colaboradores).  

O foco principal é **turnover**, analisando fatores que impactam a retenção de talentos e propondo insights estratégicos para tomadas de decisão.

Problemas identificados:

- Turnover elevado nas áreas Comercial e Tecnologia
- Alto custo de reposição
- Queda de engajamento
- Possível desigualdade em promoções

---

## 🎯 Objetivo

- Desenvolver uma base de dados simulada com problemas reais
- Realizar análise exploratória estratégica
- Conduzir testes de hipóteses
- Criar indicadores e simular impacto financeiro
- Construir modelo preditivo simples
- Gerar recomendações executivas
- Documentar resultados de forma profissional para GitHub
