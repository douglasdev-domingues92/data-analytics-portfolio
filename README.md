# Portfólio de Ciência de Dados

Este repositório reúne projetos desenvolvidos com foco em:
- **Análises Exploratórias**
- **Modelagem Preditiva**
- **NLP** 
- **Técnicas de IA Generativa**

A proposta é apresentar soluções de negócio utilizando **Python, SQL, PySpark** e **Power BI**.

## Projetos incluídos

1. **Predição de Turnover (People Analytics)**  
   Predição de turnover de colaboradores.
   
2. **Segmentação de Clientes**  
   Segmentação de clientes usando algoritmos não supervisionados.

3. **Risco de Crédito**  
   Predição de risco de crédito com modelos de classificação.   

4. **Classificação de Textos (NLP)**  
   Classificação de textos (reviews ou reclamações).

5. **Classificador de Documentos de P&D (IA Generativa)**  
   Resumo e classificação de documentos utilizando modelos generativos.

Cada pasta contém **EDA (Análise Exploratória dos Dados)**, **modelagem** (quando aplicável), **dados sintéticos** ou **públicos** e **documentação detalhada**.
